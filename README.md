# 📋 Attendance Manager

Hey there! 👋 

I built this project for my own personal use because marking student attendance manually in Microsoft Excel was taking forever, felt incredibly tedious, and was honestly just super boring. 

Instead of opening a slow spreadsheet, copy-pasting values, and looking up roll numbers row-by-row, I wanted a simple, lightweight tool that I could open instantly in any browser. This application allows me to:
- Generate a visual grid of students in two clicks.
- Paint attendance quickly by clicking buttons or typing range lists (like `1-5, 12, 18-20`) in a quick-mark box.
- Export everything directly to a clean CSV or copy a beautifully formatted text summary to clipboard in one tap.

Best of all, it works entirely offline and autosaves everything in the browser, so I never lose my active session if I accidentally refresh the page or close the tab.

---

## ⚡ How to Use

1. Download or clone this directory.
2. Double-click [index.html](file:///e:/Attendance_mark/index.html) to open it in Google Chrome, Microsoft Edge, Safari, or any other web browser.
3. Enter your starting and ending roll numbers, click **Generate Grid**, and start marking!
4. App link : https://attendance-marker-nine.vercel.app/
